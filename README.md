# İstanbul Adalet Sarayı - Dijital Yol Rehberi

Adliye içi yol tarifi, mahkeme/savcılık bulucu ve AI destekli dilekçe oluşturucu.

## Özellikler

- 🧭 **Akıllı Yol Tarifi** — Giriş kapısına göre adım adım yönlendirme
- 🏢 **Kat Rehberi** — Tüm katlar, savcılıklar, mahkemeler, bürolar
- ⚖️ **Mahkeme Bulucu** — Oda kodu ve salon numarasından arama
- 📝 **Dilekçe Oluşturucu** — 13 dilekçe türü, AI destekli
- ⚠️ **3. Kat Uyarısı** — Asansör durmaz, alternatif yol gösterir
- 📍 **4 Giriş Kapısı** — C (Vatandaş), D (Metrobüs), B (Hukuk), A (Protokol)
